🚀 Boom Net - Simulador de Atendimento ao Cliente
📖 Sobre o Projeto

O Boom Net é um sistema de atendimento via terminal desenvolvido em Python, simulando o suporte de uma provedora de internet.

O objetivo do projeto é praticar conceitos fundamentais da programação, como:

Funções
Estruturas de repetição (while)
Estruturas condicionais (if, elif, else)
Manipulação de strings
Geração de dados aleatórios
Organização de código
Menus interativos
⚙️ Funcionalidades
🛠️ Suporte Técnico
Internet lenta ou sem conexão.
Problemas para conectar ao Wi-Fi.
Abertura de chamados simulados.
💳 Financeiro
Consulta de faturas.
Consulta de negociações e acordos.
🌐 Informações da Rede
Exibição de IP público.
Velocidade da conexão.
Upload e download.
Ping da rede.
Quantidade de dispositivos conectados.
📋 Protocolo de Atendimento
Geração automática de protocolo para cada atendimento realizado.
🖥️ Tecnologias Utilizadas
Python 3
Biblioteca random
Biblioteca sys
▶️ Como Executar
Clone o repositório:
git clone https://github.com/seu-usuario/boom-net.git
Acesse a pasta do projeto:
cd boom-net
Execute o arquivo:
python main.py
🎯 Objetivo

Este projeto foi desenvolvido com fins educacionais para praticar lógica de programação e criação de sistemas interativos utilizando Python.
